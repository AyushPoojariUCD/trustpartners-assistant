# trustpartners-assistant

A production-ready RAG-based chatbot built for the Trust Partners website to provide reliable, policy-grounded guidance.

## Virtual Enviroment

`uv venv`

## Activate Virtual Enviroment

`source .venv/Scripts/activate`

## Install Dependencies

`uv pip install .`

## Run backend

`uv run uvicorn app.main:app --reload --port 8002`

## Run chatbot widget

`open chat-widget.html`
